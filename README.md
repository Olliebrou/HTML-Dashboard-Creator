# HTML-Dashboard-Creator

A browser-based dashboard builder — build, visualise, and share data dashboards without writing code.

🔗 **[Open the live app → https://olliebrou.github.io/HTML-Dashboard-Creator/](https://olliebrou.github.io/HTML-Dashboard-Creator/)**

---

The app lives in the [`chartpilot/`](./chartpilot/) directory. See its **[full README](./chartpilot/README.md)** for feature documentation, local setup instructions, and deployment notes.

## Quick start

```bash
cd chartpilot
npm install
npm run dev   # → http://localhost:5173
```

## Deployment

Pushes to `main` automatically build and publish the app to GitHub Pages via the workflow in [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml).
